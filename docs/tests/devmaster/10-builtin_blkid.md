# 特性名称

builtin blkid

## 特性描述

blkid内置命令，用于读取块设备的属性

## 特性约束

无

## 子场景 1

测试读取块设备的属性

### 备注

无

### 准备工作

无

### 测试步骤

步骤1：使用devmaster的blkid内置命令读取块设备属性
```
# devctl test-builtin blkid /sys/class/block/<device>
```

步骤2：使用udev的blkid内置命令读取块设备属性
```
# udevadm test-builtin blkid /sys/class/block/<device>
```
